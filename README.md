# Pets Home

This project is a web application for search home or hotel for pets.

## Features

- Search home or hotel for pets
- Filter by location, price, and pet-friendly amenities

## Tech Stack

- Next.js
- TypeScript
- Prisma + PostgreSQL
- NextAuth
- React Hook Form
- Redux Toolkit
